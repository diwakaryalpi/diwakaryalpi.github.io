After clicking the above link you will be directed to a page where you can login or Sign-up and then the landing page appears which displays all the contents of the application this is an ODU Library web appllication where the students can login and rent/buy a book.There are some books for free of cost so the user can rent it for free and there are some books published by ODU.The students can add the book to the cart and obtain a QR code and can directly go to the library to scan the QR code and collect the book from the front end.
